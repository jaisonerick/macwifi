# Support

Use GitHub issues for bugs, compatibility reports, and focused feature
requests.

## Good Support Requests

Include:

- What you are trying to build.
- macOS version.
- Mac model and architecture.
- Go version.
- The command or code snippet you ran.
- Full error output.
- Whether Location Services or Keychain prompts appeared.

Please anonymize SSIDs, BSSIDs, screenshots, logs, and any other private network
details before posting.

## Not Supported

This project does not provide:

- Help bypassing macOS Location Services or Keychain consent.
- Cross-platform WiFi APIs.
- Packet capture or monitor mode support.
- General WiFi troubleshooting unrelated to this library.

If you are unsure whether something belongs here, open an issue with the
smallest reproducible example you can provide.
